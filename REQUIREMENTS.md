# 小Q — 需求规格说明

> 版本：v1.5.2 | 日期：2026-06-09

## 1. 项目背景

桌面 AI 助手机器人，通过面部动画 + 语音交互提供陪伴和办公辅助。

**硬件**：树莓派 5 + DSI 屏幕（720×1280）+ 双轴舵机云台 + Hailo-8L AI 加速卡 + IMX219 摄像头 + USB 麦克风 + 扬声器
**模型**：全链路小米 MiMo — ASR(V2.5) + LLM(V2.5-Pro) + TTS(V2.5-TTS, 冰糖)
**端侧 AI**：Hailo-8L SCRFD 2.5G 人脸检测

## 3. 非功能需求

| ID | 类型 | 需求描述 | 目标值 | 当前值 |
|----|------|----------|--------|--------|
| NFR-01 | 性能 | 面部动画帧率 | 60fps | 60fps ✅ |

## 4. 已知限制

| 限制 | 说明 |
|------|------|
| 全链路需联网 | ASR/LLM/TTS 均依赖小米 MiMo API |
| MOA 需 Windows 推送 | watch_moa.js 在 Windows 上运行 |
| 仅支持中文 | ASR 模型为中文 |
| 天气/新闻 30 分钟延迟 | 后台定时采集，非实时 |
| MiMo 流式 TTS 降级 | 一 次性返回全部音频 |
