# 小Q — 需求规格说明

> 版本：v1.1 | 日期：2026-06-06

## 1. 项目背景

桌面 AI 助手机器人，通过面部动画 + 语音交互提供陪伴和办公辅助。集成 MOA 聊天、邮件知识库、待办管理、天气新闻查询，支持自然语言对话。

硬件：树莓派 5 + DSI 屏幕（720×1280）+ 双轴舵机云台 + USB 麦克风 + 扬声器

## 2. 功能需求

| ID | 需求描述 | 优先级 | 状态 |
|----|----------|:---:|:---:|
| FR-01 | 面部表情动画渲染（60fps，霓虹赛博风格） | P0 | ✅ |
| FR-02 | 语音唤醒 + ASR 识别（空格键 Push-to-Talk） | P0 | ✅ |
| FR-03 | TTS 语音播报（qwen3-tts-flash） | P0 | ✅ |
| FR-04 | NPC 人格系统（温柔陪伴等多种模式） | P1 | ✅ |
| FR-05 | Hermes Agent 集成（意图识别+技能路由） | P0 | ✅ |
| FR-06 | MOA 聊天记录查询 | P0 | ✅ |
| FR-07 | 邮件知识库查询 | P0 | ✅ |
| FR-08 | 待办事项管理（添加/提醒/完成） | P1 | ✅ |
| FR-09 | 天气查询（缓存读取，不实时 API） | P1 | ✅ |
| FR-10 | 新闻查询（缓存读取，不实时 API） | P1 | ✅ |
| FR-11 | 舵机云台控制（表情联动） | P2 | ✅ |
| FR-12 | 粒子特效 + 环境动画 | P2 | ✅ |
| FR-13 | 文字回复完整详细，语音回复简短总结 | P1 | ✅ |
| FR-14 | 人名纠错（100 人/734 条规则） | P2 | ✅ |
| FR-15 | 后台数据采集（天气/新闻，30 分钟间隔） | P2 | ✅ |

## 3. 非功能需求

| ID | 类型 | 需求描述 | 目标值 | 当前值 |
|----|------|----------|--------|--------|
| NFR-01 | 性能 | 面部动画帧率 | 60fps | 60fps ✅ |
| NFR-02 | 性能 | 语音响应延迟 | < 10s | ~7s ✅ |
| NFR-03 | 可靠性 | 崩溃自动恢复 | systemd 自启 | 手动启动 |
| NFR-04 | 可用性 | 离线时基本表情仍可用 | NPC 自主模式 | ✅ |
| NFR-05 | 安全 | API 密钥不硬编码 | 配置文件/环境变量 | ✅ |

## 4. 已知限制

| 限制 | 说明 |
|------|------|
| ASR/TTS 需联网 | 依赖阿里云 DashScope API |
| LLM 需联网 | 依赖 DeepSeek API |
| MOA 需要 Windows 推送 | watch_moa.js 需在 Windows 上运行 |
| 仅支持中文 | ASR 模型为中文 |
| 天气数据 30 分钟延迟 | 后台定时采集，非实时 |
| 新闻数据 30 分钟延迟 | 后台定时采集，非实时 |
